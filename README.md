# Accessly

Secure, role-based login → realtime chat → SSR dashboard data.

## Getting Started

### Install

```bash
pnpm i
```

### DB Setup

```bash
pnpm prisma generate
pnpm prisma migrate dev
pnpm tsx src/data/seed.ts
```

### Development

```bash
pnpm dev
```

### Testing

```bash
pnpm test
```
